# Email Validator Script

Приложение на Python для валидации и нормализации email адресов с использованием внешнего API сервиса. Полученные результаты сохраняются в CSV файл для последующей обработки.

## Описание

Приложение выполняет следующие функции:

- **Отправка запросов** к внешнему API через библиотеку `httpx`
- **Валидация данных** с помощью `pydantic` моделей
- **Преобразование email адресов** (нормализация, исправление опечаток)
- **Сохранение результатов** в файл `email_results.csv` в папке `./data/` проекта для последующей обработки
---

## Внешний API
Подробная документация по адресу: https://dadata.ru/api/clean/email/

- Проверяет формат адреса
- Исправляет распространённые опечатки
- Проверяет, не «одноразовый» ли адрес
- Классифицирует адреса на личные (@mail.ru, @yandex.ru), корпоративные (@myshop.ru) и «ролевые» (info@, support@)

Внешний API возвращает json в теле ответа такого вида:

- email: исходный email
- normalized_email: нормализованный email
- type: тип результата
- qc: показатель качества

## Конфигурация

Перед запуском приложения отредактируйте файл `main.py` и укажите email для проверки:

```python
# main.py
email: list[str] = ["example@mail.com"]
```
Для использования API необходимы ключи, которые выдаются при регистрации на сервисе на странице вашего аккаунта.
Эти ключи необходимо скопировать в значения переменных `TOKEN` и `SECRET`.
```python
# main.py
TOKEN: str = "ваш_api_ключ"
SECRET: str = "ваш_secret_токен"
```

## Установка зависимостей и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/EugeneSdrv/email-validator_script.git
   cd email-validator_script
   ```

2. **Установите зависимости вручную**
   Для установки вручную необходимо наличие uv(менеджер пакетов)
   ```bash
   uv add httpx pydantic python3.13
   ```

3. **Или используйте pyproject.toml:**
   ```bash
   uv sync pyproject.toml
   ```

4. **Запустите проект**
   ```bash
   python3 main.py
   ```

## 🚀 Установка и запуск с помощью Docker

### Дополнительные требования
   ```
   - Docker 20.10+
   ```

### Быстрый старт

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/EugeneSdrv/email-validator_script.git
   cd email-validator_script
   ```

2. **Соберите образ**
   ```bash
   docker build -t email-validator_script:latest .
   ```

3. **Запустите контейнер**
   
   для запуска контейнера с монтированием директории укажите с параметром `-v` 
   для сохранения результатов по пути `/script/data`
   ```bash
   docker run --rm --name script -v $(pwd)/result:/script/data email-validator_script:latest
   ```
